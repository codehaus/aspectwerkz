JAsCo Benchmarks: 
----------------

ant run:jasco

Important:
* Make sure that the jasco-ant.jar is not in the system classpath, otherwise the jasco compiler ant task will not work!
* Make sure that "." is in the classpath for running the benchmark.
* Make sure that javac is in the PATH environment variable.
